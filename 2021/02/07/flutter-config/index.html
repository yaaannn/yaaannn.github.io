Flutter开发环境搭建

环境搭建 VS Code Flutter
Flutter
发布日期:   2021-02-07
更新日期:   2021-02-07
文章字数:   624

1.Flutter简介

Flutter是谷歌的移动UI框架，可以快速在iOS和Android上构建高质量的原生用户界面。 Flutter可以与现有的代码一起工作。在全世界，Flutter正在被越来越多的开发者和组织使用，并且Flutter是完全免费、开源的。

2.Flutter开发环境搭建

2.1 配置Flutter SDK

1.去flutter的项目发布页或者官网下载flutter最新版。

2.将flutter解压到相关目录，我的位置是D:\flutter\

3.在用户变量中填加如下两条变量，这是flutter官方中国镜像，官网原文是这样说的：

If you’re installing or using Flutter in China, it may be helpful to use a trustworthy local mirror site that hosts Flutter’s dependencies. 

PUB_HOSTED_URL=https://pub.flutter-io.cn
FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

4.在flutter的解压目录找到flutter_console.bat并且双击运行它，第一次使用会自动下载解压相关依赖文件，这样就可以使用flutter命令了

2.2 配置Android SDK

方法同配置Flutter SDK，记得要将相关目录添加到环境变量，不会的可以自行百度

2.3 flutter doctor

命令flutter doctor用于检查flutter环境是否搭建完全，如图

如果需要，安装它们，这里的第二项由于我是使用VS Code开发，所以可以忽略

2.4 配置VS Code

在VS Code官方插件市场搜索并安装flutter插件

该插件会自动安装并启用其他相关插件

2.5 配置安卓模拟器

我这里使用的是夜神模拟器，大家也可用使用其他模拟器或者真机进行调试

打开夜神模拟器后，在命令行工具输入

adb connect 127.0.0.1:63001

出现这样结果即连接成功

至此，Flutter开发环境已经搭建完毕，下面进入Flutter实例

3.Flutter实例

1.启动 VS Code

2.打开 查看>命令面板或者使用快捷键Ctrl+Shift+P

3.输入flutter， 然后选择Flutter: New Application Project

4.指定放置项目的位置，输入项目名称， 然后回车

5.等待项目创建完成，并显示main.dart文件

6.打开 运行>启动调试或者使用快捷键F5，选择模拟器，然后就可以看到
模拟器中弹出计数器app，这是Flutter的”Hello World!”

点击右下角加号会在屏幕上反馈出来点击次数

4.总结
